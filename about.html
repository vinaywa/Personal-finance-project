ABOUT THE WEBSITE

Personal Finance is a web-application focused on helping manage day-to-day finances. It not only helps in managing transactions but also aids in managing one's savings using a table. The layout is clean and interactive for everyone to use easily. There is also graphical representation of transaction data using pie and bar graphs. Overall, it makes it easier to manage one's money in an all-time-accessible way.
